Getting started with building an Ubuntu app

We show how to get started fast and build an app from GitHub with Physalis Build Cloud. We give an overview on the structure of an Apache Cordova app and show how to install it on an ubuntu phone.

The HTML structure of an app

For the purpose of this article, we use a simple Hello World app hosted on GitHub. The HTML/JS/CSS sources are contained in www. The application settings are defined in config.xml. 

	  www/ 
	     index.html    # main-HTML file specified in config.xml
	  config.xml       # configuration file

Most importantly, the config.xml files defines the HTML files to be displayed: in this case it's index.html.

Building the app

We can build an app by installing cordova and creating a click chroot for armhf like explained in the Cordova Guide at ubuntu developer. It's however much easier to use Physalis.

First a create an Account at Physalis Build Cloud (if you haven't already done so). Then click on "create a new project". In the following form, enter the project name, the following git URL https://github.com/physalisio/helloworld.git and click "import project!".

A project box appears, where you can press "Pull & Build". Afters some times (20-40 seconds) The build is finished. You can either download the app binary or inspect the build log file.
